Demo Script: SoloBob Supply Chain Copilot
1. Opening
Hi, I am presenting SoloBob Supply Chain Copilot, a domain-specific AI decision-support system for supply chain managers built on Snowflake.
The problem is that managers often need to make inventory decisions quickly, but the relevant information is spread across stock, sales, supplier, and delivery data. SoloBob brings these signals together and turns them into risk alerts and replenishment actions.
2. Show Dashboard
This dashboard monitors five products and classifies them into high, medium, or low risk.
The KPIs show:
Number of products monitored
High-risk products
Medium-risk products
Low-risk products
Total urgent reorder quantity
3. Run Workflow
Click Run SoloBob Copilot.
This executes the Snowflake procedure `RUN_SOLOBOB_COPILOT()`. It refreshes action logs, creates critical alerts, stores run history, and prepares a manager brief.
4. Explain Risk Dashboard
The Risk Dashboard shows product-level analysis using current stock, safety stock, reorder point, average weekly sales, supplier reliability, and delivery delay.
5. Explain Recommended Actions
The Recommended Actions tab turns analysis into decisions. It tells the manager which products require urgent replenishment and how many units to order.
6. Explain Critical Alerts
The Critical Alerts tab shows urgent risks generated by the workflow. These alerts are stored in Snowflake for traceability.
7. Explain Supplier Concerns
The Supplier Concerns tab identifies suppliers with low reliability and delivery delays. This helps managers understand root causes, not just symptoms.
8. Explain Agent Skills
SoloBob uses three modular agent skills:
Inventory Risk Analysis Skill
Supplier Delay Intelligence Skill
Replenishment Recommendation Skill
This makes the copilot domain-specific rather than a generic chatbot.
9. CoCo / Cortex Code Demo Questions
Ask Snowflake CoCo / Cortex Code:
Which products are at highest inventory risk?
Why is Bluetooth Speaker considered high risk?
Which products should we reorder first?
Give me a manager summary of current supply chain risks.
Which supplier is causing the most operational concern?
10. Closing
SoloBob Supply Chain Copilot demonstrates how Snowflake can support an end-to-end AI-native workflow that moves from enterprise data to risk reasoning, recommendations, alerts, and manager-ready decisions.
